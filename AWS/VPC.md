# VPC学習記録

## 実施したこと
- VPCの構成を完成図通りに作成する
- VPCとサブネットを作成（パブリックサブネットとプライベートサブネット）
- パブリックサブネット上にEC2インスタンスを作成、Apacheをインストール
- パブリックサブネット上にNAT GWを構築

## VPCとは
- クラウド上に仮想的なネットワークを構築するためのサービス

## プライベートサブネットとパブリックサブネット
- パブリックサブネットはインターネットと直接通信できるサブネット
- プライベートサブネットはインターネットと直接通信できないサブネット

## ルートテーブルとは
- VPCにおいてルーティングによるサブネット外への通信経路制御を実現
- 通信先を0.0.0.0/0にすると外部のインターネットと直接通信できるようになる←パブリックサブネットになる


## 現時点での理解レベル
- VPCは「街」のイメージ、サブネットは「建物」でEC2などは「部屋」と理解
- ルートテーブルは「地図」のイメージ。普通にルーティングをしていると解釈
- 業務で使用している「踏み台サーバー」ってそういうことだったのか、外部から直接通信できないのか
